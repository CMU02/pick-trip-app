# Pick Trip App

## 프로젝트 소개

Pick Trip App은 하동, 영주, 예천처럼 상대적으로 덜 알려진 지역의 음식, 축제, 관광지, 문화 및 자연 콘텐츠를 소개하는 국내 여행 일정 생성 서비스입니다.

사용자가 원하는 지역 콘텐츠를 직접 선택하면, AI가 여행 날짜와 동행 조건에 맞춰 현실적인 여행 일정을 구성합니다. 단순한 여행지 추천을 넘어 사용자가 고른 콘텐츠를 실제 이동 동선과 일정으로 연결하는 것을 목표로 합니다.
