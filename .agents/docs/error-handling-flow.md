# 예외 처리 흐름

## 1. 로그인 / 인증 실패

- **원인**: Kakao/Google OAuth 서버 오류, 네트워크 단절
- **처리**: 오류 메시지 표시 후 로그인 화면 유지
- **메시지 예시**: "로그인에 실패했습니다. 잠시 후 다시 시도해주세요."

## 2. 컨텐츠 로드 실패

- **원인**: TourAPI 응답 오류, 서버 오류, 네트워크 단절
- **처리**: 빈 목록 대신 오류 안내 UI 표시, 재시도 버튼 제공
- **메시지 예시**: "컨텐츠를 불러오지 못했습니다. 다시 시도해주세요."

## 3. AI 일정 생성 실패

- **원인**: AI 서버 응답 오류, 타임아웃, 입력 데이터 부족
- **처리**:
  - 타임아웃: 일정 생성 중 로딩 표시 후 실패 시 재시도 유도
  - 입력 부족: 컨텐츠를 더 추가하도록 안내
- **메시지 예시**: "일정 생성에 실패했습니다. 컨텐츠를 추가하거나 다시 시도해주세요."

## 4. 일정 저장 실패

- **원인**: 서버 오류, 네트워크 단절, 인증 만료
- **처리**: 저장 실패 알림, 재시도 버튼 제공. 인증 만료 시 재로그인 유도
- **메시지 예시**: "저장에 실패했습니다. 다시 시도해주세요."

## 5. 일정 공유 실패

- **원인**: 공유 링크 생성 서버 오류
- **처리**: 공유 실패 알림, 재시도 버튼 제공
- **메시지 예시**: "공유 링크 생성에 실패했습니다. 잠시 후 다시 시도해주세요."

## 6. 네트워크 단절 (공통)

- **처리**: 오프라인 상태 감지 시 상단 배너로 안내
- **메시지 예시**: "인터넷 연결을 확인해주세요."
