# MVP 범위

초기버전은 기능을 과도하게 넓히기보다는 3개 지역의 핵심 경험을 완성하는 데 집중한다.

## MVP 포함 기능

- [ ] 로그인 / 회원가입
  - 조건: Kakao 또는 Google 소셜 로그인으로 정상 인증 가능
- [ ] 하동, 영주, 예천 지역 선택
  - 조건: 3개 지역 중 1개 이상 선택 가능
- [ ] 여행 날짜와 기간 설정
  - 조건: 당일치기~커스텀 기간 설정 가능
- [ ] 동행 조건 선택
  - 조건: 1개 이상의 동행 조건 선택 가능
- [ ] 지역 콘텐츠 카드 탐색 및 검색
  - 조건: 카테고리 필터 및 검색으로 컨텐츠 탐색 가능
- [ ] 콘텐츠 담기
  - 조건: 1개 이상의 컨텐츠를 여행 바구니에 추가 가능
- [ ] 우선순위 설정
  - 조건: 담은 컨텐츠에 꼭 가기 / 가면 좋음 / 시간 남으면 설정 가능
- [ ] AI 일정 생성
  - 조건: 2개 이상 컨텐츠 선택 시 1박 2일 기준 일정 1개 이상 생성 가능
- [ ] 일정 수정
  - 조건: 장소 순서 변경, 삭제, 대체 장소 추가 가능
- [ ] 일정 저장 (서버 저장)
  - 조건: 로그인 후 저장한 일정이 앱 재시작 후에도 조회 가능
- [ ] 일정 공유
  - 조건: 저장된 일정을 링크 또는 공유 기능으로 외부 공유 가능

## MVP 제외해도 되는 기능

- 숙소 예약
- 결제
- 실시간 교통
- 사용자 커뮤니티
- 리뷰 시스템
- 모든 경상도 지역 확장
- 완전 자동 데이터 수집
